# Data Transfer Objects

DTOs for API request/response communication.

## Subfolders
- `Auth/` - Authentication DTOs (Login, Register, etc.)
- `User/` - User-related DTOs
- `Template/` - Template DTOs
- `Invitation/` - Invitation DTOs
- `Public/` - Public API DTOs
- `Upload/` - File upload DTOs
- `Admin/` - Admin operations DTOs
